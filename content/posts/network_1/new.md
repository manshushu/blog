# 其中重要的知识

## 路由器
1. 操作系统叫ios
2. nvram断电还有数据
3. 路由器启动步骤
> 自检（power-on self test），引导装入程序
> 定位加载 ios（**可以是云服务器上的系统**） 定位ios 加载ios
> 定位加载配置软件

4. 路由器的硬件组成：
- cpu
- ram
- flash（硬盘）
- nvram
- rom
- 各种接口

5. 基本路由器设置：
  - consle口（双反线）
  - 串口（一边有时钟一边没时钟）
  - 思科系统的模式
    - 用户（最低级）
    - 特权
    - 全局
    - 接口
    - 子接口
    - 行模式
