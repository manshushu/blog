# 必带

- 床上的
- 衣服
- 电脑
- 鼠标
- 手柄
- 充电器
- 鞋子！！！

# 不必要带
- 书
- 床垫
- 被子


# 用什么带
- 提包带衣服和部分床上用品
- 书包带电脑和书
- 床上用品包带床上用品

# 除了带东西还要注意什么
- 放空饮水机
- 吧蛋白粉拿出来
- 吧除湿的全部拿出来
- 断电
- 扫地
- 大麦茶丢掉
- 药带回家
